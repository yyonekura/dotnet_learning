C#の構文の理解を進める

1日目と同様、実際にコードを動作させながらC#の構文の理解を勧めましょう。
2日目に用意した実行環境か、Try DOTNETを利用して動作を確認します。

条件分岐文

条件分岐文ifは、プログラム中の条件によって処理を変更したい場合に使用します。 


// 数値の入力を受け付ける
string inputText = Console.ReadLine();
int number = int.Parse(inputText);

// numberが偶数であるかを判断
if(number % 2 == 0){
    Console.WriteLine("偶数です");
}
else {
    Console.WriteLine("奇数です");
}

条件分岐文switchは、一つの変数の中身が複数通りに分岐するようなプログラムを書く際に便利な構文です。 


int caseSwitch = 1;

switch (caseSwitch)
{
    case 1:
        Console.WriteLine("Case 1");
        break;
    case 2:
        Console.WriteLine("Case 2");
        break;
    default:
        Console.WriteLine("Default case");
        break;
}

配列を理解する

配列は、変数を複数個まとめて扱うことができる構文です。
各曜日の名前を一括して扱ったり、日付ごとの売上を順番に処理したりと、繰り返し処理したいデータを格納するのに向いています。
次の項目で説明する繰り返し文といっしょに使うことで、あらかじめ処理するデータを配列に用意して、それを順に繰り返して処理することができます。

繰り返し文

繰り返し文forは、プログラム中で何度も繰り返す処理を記述する際に利用します。 


// 10回繰り返す。
// セミコロン区切りで、それぞれ
//（最初の一回のみ実行）；（条件式が成立する限り繰り返す）；（一周ごとの処理）
// を意味します。
for(int i = 0 ; i < 10 ; i++){
    Console.WriteLine(i);
}

繰り返し文foreachは、あらかじめ決まったデータの集まりがあり、それを1件ずつ処理する場合に便利な繰り返しの構文です。
下記の例では、変数の配列に入った一連の数値を一つずつ取り出して出力しています。 


int[] numbers = { 4, 5, 6, 1, 2, 3, -2, -1, 0 };
foreach (int i in numbers)
{
    System.Console.Write("{0} ", i);
}

ジャンプ文

プログラムの実行中、特定の位置に処理の順番を移動する（ジャンプする）時に使用する構文です。
下記の例では、continue文でfor文の先頭に戻っています。 


// 0から9の数のうち、奇数のみ出力する。
for(int i = 0 ; i < 10 ; i++){
    if(i % 2 == 0){
        // continueにたどり着いたら、このループを終了してforの先頭に戻る
        continue;
    }
    Console.WriteLine(i);
}

基本データ型を理解する・整数型と実数型

数値型には、整数型と実数型が存在します。
整数型は小数点以下を持たない数のカウンタや物品の数等に使用することになります。
実数型（浮動小数点型）は温度や長さ、重さ等整数のみでは扱えない各種データを格納するのに使用します。

例として、下記のように実数型と整数型では格納可能な値に違いが出ます。 


// 実数の値を代入
double doublenumber = 3.14;

// 整数型に変換する
// 小数点以下は切り捨てられる
int intnumber = (int)doublenumber;

Console.WriteLine(doublenumber);
Console.WriteLine(intnumber);

基本データ型を理解する・論理型

論理型は、真偽（trueとfalse）の二種類だけを持つデータ型です。
先述したif文は、括弧の中の真偽値がtrueかfalseかによって、処理を分岐しています。
if文の条件に利用する比較式は、条件が成立していた場合にはtrueを、そうでなければfalseを結果として返す動作をします。 


// 比較式が正しければTrue、誤りであればFalseが表示される
Console.WriteLine(1 + 2 == 3);
Console.WriteLine(10 + 1 == 1);

// 論理型はbool型の変数に保存できる
bool check = 1 + 2 == 3;
Console.WriteLine(check);

算術演算子を理解する

算術演算子は、加減乗除をプログラミング言語上で行う為の演算子です。
下記の例では簡単な加減乗除の計算を行い、出力しています。 


int a = 5;

a = a + 9;
Console.WriteLine(a);  // 答え: 14

a = a - 4;
Console.WriteLine(a);  // 答え: 10

a = a * 2;
Console.WriteLine(a);  // 答え: 20

a = a / 4;
Console.WriteLine(a);  // 答え: 5

a = a % 3;
Console.WriteLine(a);  // 答え: 2

算術演算子は、=と複合して計算と代入を同時に行うことができます。
下記の例は、上記の例と全く同じ意味を持ちます。 


int a = 5;
a += 9;
Console.WriteLine(a);  // 答え: 14

a -= 4;
Console.WriteLine(a);  // 答え: 10

a *= 2;
Console.WriteLine(a);  // 答え: 20

a /= 4;
Console.WriteLine(a);  // 答え: 5

a %= 3;
Console.WriteLine(a);  // 答え: 2

算術演算子には、途中経過に括弧をつけることで計算の優先順位を変更することができます。 


Console.WriteLine(2 + 2 * 2);   // 答え: 6
Console.WriteLine((2 + 2) * 2); // 答え: 8

Console.WriteLine(9 / 5 / 2);   // 答え: 0
Console.WriteLine(9 / (5 / 2)); // 答え: 4
